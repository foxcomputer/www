# foxcomputer Website

Modern, open-source website for foxcomputer, built with React 19, Tailwind CSS v4, and Shadcn/UI.

## Tech Stack

- **React 19** with TypeScript
- **Tailwind CSS v4** - Latest CSS-first approach
- **Shadcn/UI** - Neutral theme components
- **Vite** - Fast build tool

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

This site is automatically deployed to Vercel on push to main.
